📐 Fórmula de Bézier

B(t) = (1-t)³·P₀ + 3(1-t)²t·P₁ + 3(1-t)t²·P₂ + t³·P₃
0 ≤ t ≤ 1

Como usar:
1. Clique no canvas para adicionar até 4 pontos de controle
2. Arraste os pontos para modificar a curva
3. Ajuste os segmentos para suavidade

Limites: X[-255, 255], Y[-255, 255]
